# 엘리베이터 시뮬레이터 기획서

## 개요
건물 내 엘리베이터의 운행을 시각적으로 시뮬레이션하는 인터랙티브 웹 애플리케이션.
다양한 스케줄링 알고리즘(FCFS, SCAN, LOOK, SSTF)을 선택하여 운행 효율 차이를 관찰.

## 핵심 기능
1. 10층 건물(B1~9F) 세로 단면도 시각화
2. 엘리베이터 캐빈 CSS transition 이동 + 도어 애니메이션
3. 각 층 ▲/▼ 호출 버튼, 내부 패널 층 선택 버튼
4. 4종 스케줄링 알고리즘 (FCFS, SCAN, LOOK, SSTF)
5. 자동 승객 생성 (토글 + 빈도 슬라이더)
6. 실시간 통계 (평균 대기/이동 시간, 운송 인원, 이동 거리)
7. 시뮬레이션 속도 제어 (x1, x2, x5, x10) + 일시정지

## 기술 제약
- file:// 동작, fetch 금지, 상대 경로만 사용
- index.html + style.css + app.js 3파일 분리
- localStorage 키: `hs_elevator_` 접두사
